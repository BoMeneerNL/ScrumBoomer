java

welkom op de pagina over java. Hier wordt uitgelegd wat Java is, waarom de leerlingen het krijgen en wat het nut is van deze programmeertaal.
Er wordt ook uitgelegd welke vorm van toetsen we hiermee hanteren. Java is een programmeertaal die uit kwam begin jaren 90 en is ontworpen door James Gosling. java is een platformonafhankelijke programmeertaal als een programmeertaal platformonafhankelijk is betekend het dat het uit bytecode bestaad. een codesoort die tussen broncode en machinecode inzit en moet voor het uitvoeren vertaald worden. En omdat het platformonafhankelijk is kan het op elke willekeurige computer draaien dus is het een handige programmeertaal om te leren.

Waarom wordt Java op het ROC gegeven?

Java is een van de moeilijkere programmeertalen die gegeven wordt op het ROC. Het is vooraal handig om je meteen te verdiepen in de stof de gegeven word. (hiermee wordt vooral de basis van java bedoeld) Zoals de termen, begrippen, nieuwe structuren, allerlei vreemde en nieuwe tekens.

Wat is het voordeel van Java leren / gebruiken?

Een groot voordeel als je Java als OOP (Object-Oriented Programming een programmeertaal die gebruikt maakt van objecten in plaats van acties. en gebruik maakt van gegevens in plaats van logica om applicaties en computerprogramma's te ontwerpen. (tegenwoordig ondersteunen de meetse programmeertalen OOP.)) leert en begrijpt, zul je zien dat het leren van een andere programmeertaal je heel makkelijk afgaat. Dit is eigenlijk het grootste voordeel. Naast dat wordt Java ook bij veel verschillende bedrijven gebruikt, waardoor het zoeken naar een baan gemakkelijk afgaat.

waarvoor wordt Java gebruikt?

Java woord voornamelijk gebruikt voor apps/computerspellen denk bijvoorbeeld aan Minecraft of je mobiel zoals android, althans is de heele OS gebaseerd op java zelf. Er worden ook opensourcebibliotheken ontwikkeld, zoals de Lightweight Java Game Library (LWJGL) en jMonkeyEngine. Lightweight Java Game Library is een open source programma waarin je jouw eigen game in kan maken. En jMonkeyEngine is een engine voor het ontwikkelen van 3D-computerspellen in Java.

Hoe worden de toetsen van Java afgenomen?
