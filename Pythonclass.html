Python

welkom op de pagina over python. Hier wordt uitgelegd wat python is, waarom de leerlingen het krijgen en wat het nut is van deze programmeertaal.
Er wordt ook uitgelegd welke vorm van toetsen we hiermee hanteren. Python is een best oude programmeertaal die veel wordt gebruikt. Het bestaat sinds de jaren 90.
Het is ontwikkeld in het centrum voor Wiskunde en Informatica in Amsterdam.
Het was ontwikkeld door een groep mensen waar ook de professor Lambert Meertens deel van uitmaakte.
Meneer van Rossum had samen met deze enthoursiaste groep tot 2018 dit project geleid. Deze groep werd ondersteunt door vrijwilligers op het internet
De ontwikkeling wordt nu geleid door de Python Software Foundation. Het is een Open Source Programmeertaal.

Waarom wordt python op het ROC gegeven?

Python is een heele uitgebreide computertaal en kan voor heel veel dingen gebruikt worden zoals webdevelopment, videogames, apps, simple scripting, data analyse, machine learning.
het kan ook gebruikt worden als een extensietaal voor applicaties die geschreven zijn in een andere programmeertaal.

Wat is het voordeel van python leren?

Het grootste voordeel van Python is dat het zo uitgebreid is en kan gebruikt worden voor heel veel verschillende functies.
Het is overzichtelijk en makkelijk te begrijpen.

Hoe worden de toetsen voor python afgenomen?

Je krijt een project van Python waarin je een bepaalde website in een bepaalde tijd moet maken. en die wordt dan beoordeeld.
